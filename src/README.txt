D{0-1}KP Tkinter 图形界面程序使用说明
======================================

一、运行环境
------------
Python 3.10 及以上版本

依赖安装：
pip install matplotlib numpy

Tkinter 一般为 Python 自带库，无需额外安装。

二、文件结构
------------
main.py            程序入口
models.py          数据模型定义
parser_utils.py    数据文件解析
solver.py          排序与动态规划求解
plot_panel.py      Matplotlib 绘图面板
main_window.py     Tkinter 主窗口界面

三、运行方式
------------
在项目目录下执行：
python main.py

四、功能说明
------------
1. 打开 txt 数据文件，自动解析多个 IDKP 实例
2. 在左侧列表中选择实例
3. 可绘制当前实例的重量-价值散点图
4. 可按第三项价值/重量比排序并显示表格
5. 可用动态规划求解当前实例的最优解
6. 可将求解结果保存为 txt 文件

五、问题建模
------------
D{0-1}KP 可视为一种分组背包问题：
- 每个项集包含 3 个候选物品
- 每组至多选择 1 个物品装入背包
- 在总重量不超过容量约束下，使总价值最大

本程序采用动态规划进行精确求解。
