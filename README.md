# D{0-1}KP 求解系统
软件工程个人项目｜基于动态规划的 D{0-1} 背包问题求解工具

## 功能
- 读取 D{0-1}KP 数据文件
- 绘制重量-价值散点图
- 按项集第三项价值重量比排序
- 动态规划求解最优解
- 结果导出 TXT/Excel
- 图形化交互界面

## 环境
Python 3.8+  
依赖：matplotlib pandas openpyxl

## 运行
```
python src/main.py
```
